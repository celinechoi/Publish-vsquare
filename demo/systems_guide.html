VSQUARE DEMO 프로젝트의 systems.scss 파일 기준으로 작성하였습니다.

  • VSQUARE DEMO 프로젝트 퍼블 주소 : https://publish.vsquare.cc/manage/sub/demo_v2.html
  • Zeplin의 🔸VSQUARE > Design System Kit와 함께 업데이트 될 예정이며,
    최신 systems.scss를 🔗https://github.com/vsquaregit/Publish-DEMO_v2/blob/main/style/systems.scss에서 다운받아 사용할 수 있습니다.
  • 본 페이지에 외의 요소 설명은 systems.scss 파일 안의 주석을 확인바랍니다.

SCSS/CSS (위치 : "style/" 또는 "icon_fonts/")

  • 변경가능 파일 : var_function.scss (// @개인 변경 가능 주석 아래 부분 해당 - 프로젝트 폰트, Primary, Secondary, 미디어 쿼리 변수)/ style.scss (프로젝트별 스타일 커스텀)
  • 고정 파일 : normalize.scss systems.scss icon_fonts/icon_fonts.min.scss

작업자는 style.scss과 var_function.scss 파일을 편집하여 CSS 작업을 하며, 최종적으로 "style/style.min.css" 과 "icon_fonts/icon_fonts.min.css"를 연결하게 됩니다.
(그 외의 프로젝트별 연결되는 플러그인의 스타일이 추가 될 수 있음)

예시 텍스트들은 "해당 스타일의 클래스명" 입니다.

변수 선언

위치 : var_function.scss // @개인 변경 가능 OOOOOOOOOOOOOOOOOOOOOO

  • $font-- : 해당 프로젝트의 폰트 패밀리 값을 작성합니다.
  • $primary-- : 해당 프로젝트의 Primary 색상 값을 작성합니다.
  • $secondary-- : 해당 프로젝트의 Secondary 색상 값을 작성합니다.
  • @mixin tablet / @mixin mobile : 해당 프로젝트의 Tablet과 Mobile의 max-width 값을 지정합니다.

타이포

위치 : systems.scss // @Typo

고정된 값으로 Design Systems Guide Kit(V2.2) 2) Typography에 있는 "Headline"과 "Subtitle"를 ".txt--b"와 ".txt--m"으로 시작하는 클래스로 정하였습니다.
systems.scss 안의 주석 // @Typo를 참고하여 사용합니다.

  • 11 ~ 16 까지는 1단위로 나눠지며, 16 ~ 38 까지는 2단위로 나눠집니다.
  • 폰트 스타일에 따라 두께와 크기가 구분됩니다.
  • 타이틀 영역 (Bold/Medium)은 1.4em 값을 사용합니다. (글자크기 x 1.4 = 행간)
  • 기본 폰트는 폰트 크기에 비례한 1.75em 값을 사용합니다. (글자크기 x 1.75 = 행간)

  • 기본 span태그를 사용하여 마크업을 합니다. 링크가 있는경우, a태그(href="javascript:;")로 연결합니다.
