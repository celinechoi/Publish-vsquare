오시는길

한신대학교 경삼관 1층 대학일자리플러스센터

031-379-0055

  • 대중교통 이용시
  • 자가용 이용시

전철 이용시

1호선 병점(한신대)역 하차, 2번출구 버스정류장에서 마을버스(55,56) 이용

시내버스 이용시

  • 마을08 한신대 후문 하차 후 도보이동

    1. 수원에서 오는 길 : 망포역 → 병점사거리 → 한일타운 → 한신대 후문
    2. 오산에서 오는 길 : 오산역 → 세마역 → 한신대 후문
    3. 한신대 후문 하차 → 송암관까지 도보 5-7분 소요

  • 마을07 효성아파트 후문 하차 후 도보이동

    1. 병점에서 오는 길 : 롯데시네마 → 병점사거리 → 효성아파트 후문
    2. 오산에서 오는 길 : 오산역 → 세마역 → 효성아파트 후문
    3. 효성아파트 후문 하차 → 송암관까지 도보 5-7분 소요

  • 병점사거리 환승 가능 버스

    1. 병점에서 오는 길 : 롯데시네마 → 병점사거리 → 효성아파트 후문
    2. 오산에서 오는 길 : 오산역 → 세마역 → 효성아파트 후문
    3. 효성아파트 후문 하차 → 송암관까지 도보 5-7분 소요
